 BeautyJson is a tool for javascript object format,it is easy to make json readable.<br>
 Usage : <br>
     beautyJson({id:1,name:'Owen',history:[{date : new Date(),b:true}]}})<br>
   Output :<br>
    { <br>
   	  "id" : 1, <br>
   	  "name" : "Owen", <br>
   	  "history" : [{ <br>
   	    "date" : "2012-10-19 15:40:41", <br>
   	    "b" : true<br>
   	  }]<br>
   	}<br>
